# 开始卷题——题目

![](http://www.ylcourt.cn/api/public/img/027d992f92d142c0be1bb0dfa48bb8fd.png)

## 总览

这里是云兰阁的核心板块。在云兰阁中，目前已经有了上千道题目。题目分为多个题库：主题库    HDU Codeforces POJ GYM AtCoder SPOJ BAS。目前，云兰阁导入了Codeforces的题目以及BAS的题目（BAS题目就是xyxy后来增加的入门题目）。给大家提供了丰富的题目。

同时，我们对题目难度进行了划分：**入门 普及- 普及/提高- 普及+/提高 提高+/省选- 省选I/NOI+/CTSC**。方便同学按照自己的能力水平高效科学刷题。

在题库的右侧，还有题目的统计信息，以及标签系统。可以快速筛选题目。

# 成果的检验——比赛

## 比赛类型

比赛类型分为**ACM和OI**，这里的ACM和OI与xyxyOJ不同。ACM没有分数机制，只看做题数量和罚时；而OI赛制有分数，以分数为主要排名。绝大多数我们使用OI赛制。

比赛分为**公开赛，保护赛，私有赛**。

公开赛：不需要密码直接参加。

保护赛：可以查看题目，但提交需要密码。

私有赛：需要密码进入。

## 开始比赛

比赛界面分为为 **比赛简介、题目、状态、排名、公告、评论**。

### 比赛简介

这里会展示比赛的相关信息，在比赛前请认真阅读。

### 题目

在这里会显示比赛的题目也会展示题目的通过率。

### 状态

这里相当于提交记录，你可以看到自己在比赛中所有的提交记录。

### 排名

这里会有排行榜，可以快速查看自己在比赛中的位次以及他人的做题情况。

### 公告

这里会显示比赛的公告，请留意。

### 评论

你可以在这里询问比赛相关问题，**不可询问解法等以及无意义内容**。

# 登上巅峰——排名

排名分为**OI排名**和**ACM排名**。

ACM排名只看做题数目，在实际学习中的参考价值不高，且功能与OI排名基本一致，这里不再阐述

## 图表

在这里会将每个人的分数以**折线图和柱状图**来呈现，也可以在这里下载图表。

## 主区域

排名系统中包括**用户，昵称，真实姓名，分数，AC/总数，通过率，个性简介**。

### 用户&&昵称&&真实姓名

这里是每个用户的基本信息，除用户名之外，昵称和真实姓名都可以在个人设置中进行修改，但是为了方便统计信息，**请不要对真实姓名进行修改**。

### 昵称&&分数

每AC一道题目，OI排行中就会增加相应的分数，具体的规则为：

$0.1\times 分数+2 \times 难度$

在题目中，可以查看每道题你可以获得的分数。

**那分数有什么用呢？**

#### 名字颜色系统

根据不同的分数，你的昵称颜色也会改变。~~一定程度上，名字颜色可以反应你的实力~~。

| 颜色 | 分数 |
| ---- | ---- |
| 黑名 | 6000 |
| 红名 | 3000 |
| 橙名 | 2000 |
| 绿名 | 1500 |
| 蓝名 | 500  |
| 灰名 | 0    |

除此之外，还有管理员特有的紫名，这是管理员的象征，以及臭名昭著的棕名，棕名用于**处罚在云兰阁中违反管理规定的情况**。

**请注意，名字颜色不能反映一切实力，不要过于看重名字颜色而做一些不符合规定的事情**

### tag

tag是在名字后面添加的一个牌子，~~最常见的牌子是**作弊者tag**~~。

因云兰阁还没有确认tag的具体用法，这里不多阐述。

# 强大的社区功能——讨论

在讨论中，有多个板块，如：闲聊，题解，求助，建议，记录等等。

### 闲聊

~~在这里你可以尽情摸鱼~~，但请记住，云兰阁是一个学术类型偏多的OJ，不要把时间浪费在无意义内容上。此板块管理员会定期清理无意义帖子。

### 题解

在这里会有许多巨犇发布的题解，会对我们的学习有很大帮助，**为了减少抄袭现象，云兰阁中大部分题解不展示完整代码**。此外，此区域**禁止发布无意义的，与题目无关的内容**，如果多次违反，将禁止此用户发帖。

### 求助

在这里你可以求助一些不懂的问题，包括但不限于**OI，数学**等问题，但是不可以在这里发布除此之外的内容，违反者同上。

### 建议

在这里你可以反馈一些云兰阁中的问题，如题目排版，题目错误，新需求等等。但是如果发现云兰阁有重大bug，请不要发帖，直接联系**EBeason或Introl**。

### 记录

这种类型的记录模块旨在帮助编程爱好者或竞赛选手追踪他们在解决编程问题方面的进度和表现。

# 团队

## 类型

公开团队：可以直接加入。

保护团队：可以申请加入。

私有团队：需要密码才可以申请。

## 用途

团队主要用于教师教学使用，在团队中相当于独立的一套系统，与外界基本相同，不在阐述。

# 个人设置

这个板块会讲述如何正确设置个人信息。

## 账户设置

在这里你可以更改密码，邮箱，~~手机号~~（不可使用）

### 更改密码

你可以在这里修改自己的密码，**一定一定要记牢**。

**在获取到自己账号的随机密码后，请立即更改密码**

### 更改邮箱

你可以在这里输入自己的邮箱。

首先输入自己的密码确认身份，然后输入你的邮箱，点击右侧的绿色按键，这时会有一封包含验证码的邮件发到你的邮箱，将验证码在文本框中输入即可绑定。

**邮箱可以帮助你在忘记密码时进行修改，十分重要，建议设置**

### 更改手机号

云兰阁目前暂不支持。

## 资料设置

在这里你可以修改对外展示的个人信息。

### 头像

拖动或者选择图片到此处可以更改自己的头像。

### 真实姓名

**请确保此为你的真实姓名，不要对他进行修改，否则数据统计时会出现问题。**

### 呢称

可以自己设置自己喜欢的昵称。但是**建议不要使用空白昵称，请保持他默认与用户名相同**。

### 学校&&学号

可以设置自己的学校，设置后会在某些排名和主页显示

### codeforces用户名

你可以在这里填入自己的cf用户名，他会同步你的CF Rating。

### 博客

在这里可以添加自己的博客链接。注意要放入完整链接。

### GitHub

同样需要放入完整跳转链接。

### 个人简介

你可以在这里填写个人简介，支持**Markdown渲染**。
